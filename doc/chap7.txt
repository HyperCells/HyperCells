  [1X7.1 [33X[0;0YPoint-group matrices[133X[101X
  
  [33X[0;0YPoint-group  matrices  are  representations  of  group elements in the point
  group  [23XG\cong\Delta/\Gamma[123X.  These  matrices can efficiently be constructed.
  Given  any  symmetry  [10Xg[110X  in the point group [23XG[123X and a set [10XI[110X of all translation
  generators  [23X\gamma_i[123X  of  the  translation  group  [23X\Gamma[123X,  a  corresponding
  point-group  matrix  can  be  determined  through  the  set  of the symmetry
  transformed elements [23Xg \gamma_i g^{-1}[123X [CGL+23]. Point-group matrices enable
  the conduction of symmetry analysis on any given (supercell) model graph.[133X

  [1X7.3-1 PGMatrices[101X
  
  [33X[1;0Y[29X[2XPGMatrices[102X( [3Xsymmetries[103X, [3XpgMatsGs[103X ) [32X function[133X
  [6XReturns:[106X  [33X[0;10Ylist of point-group matrices as a [10XPGMatrices[110X object.[133X
  
  [33X[0;0YConstructs  point-group  matrices for a single symmetry element or a list of
  symmetry elements given as words written in terms of the rotation generators
  [10Xa[110X, [10Xb[110X, [10Xc[110X of the triangle group [23X\Delta[123X or the reflection generators [10Xx[110X, [10Xy[110X, [10Xz[110X of

  [10Xfulltg[110X  and  [10XProperTriangleGroup[110X  object  [10Xtg[110X,  respectively  (see [14X2.1[114X). Each

  representations  of  the point group previously specified by a corresponding
